## ✨ About

**Spotify Clone** is a Spotify-inspired web music player interface built using **HTML5** and **CSS3**. The project recreates the look and feel of Spotify with a modern dark theme, responsive layout, sidebar navigation, playlists, albums, and music player controls.

It focuses on front-end development, responsive design principles, and UI/UX implementation while closely mimicking Spotify's visual experience.

---

## 🛠️ Tech Stack

* HTML5
* CSS3

* ✅ Spotify-inspired user interface
* ✅ Responsive dark theme design
* ✅ Sidebar navigation menu
* ✅ Playlist and album sections
* ✅ Music player layout
* ✅ Modern UI components
* ✅ Mobile-friendly interface

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/yashig235/Spotify-Clone.git

# Open project folder
cd Spotify-Clone
```

### ▶️ Run the Project

Simply open:

```bash
clone.html
```

in your browser.

You can also use the **VS Code Live Server Extension** for a better development experience.
